<h1>Dessert Web App</h1>

A web application that allows to sell and order desserts.

## View Site

Open [Dessert](https://yj-desserts.herokuapp.com/) with your browser to see the result.

## Features

- Signin / Signup
- Posting
- Search
- Add to cart
- User profile: posts, cart

## How to use

To run this project, install it locally using npm:

```bash
npm install
npm start
```
